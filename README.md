### 개발환경
- **Language**: java 11, javascript, html 5, css 3
- **IDE**: IntelliJ Ultimate
- **Framework**: SpringBoot 2.7.9
- **Build**: Gradle
- **DB**: 
- **ORM**: JPA
- **Library**: jquery, bootstrap

### 주요기능   
 - **게시글**
